# RCAccelerator

## RCAccelerator: What is it?

A conversational tool to expedite a Root Cause Analysis (RCA). An interactive artificial intelligence tool called RCAccelerator helps engineering teams find, examine, and link failures to Jira-documented problems. Based on [Chainlit](https://www.chainlit.io/), it matches user queries by using LLMs and searches in a preprocessed Jira ticket inventory.


## How It Operates

1. Recognizes user inquiries to identify similar issues in Jira tickets.
2. Generates a response that summarizes similar issues found in the previous step.
